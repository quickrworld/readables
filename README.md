# Readable App

To get started right away:

* Install and start the API server
    - `cd myworkdir`
    - `git clone https://github.com/quickrworld/readables.git`
    - `cd api-server`
    - `npm install`
    - `node server`
* In another terminal window,
    - `cd myworkdir/readables`
    - `cd frontend`
    - `npm install`
    - `npm start`
